# Orange Web EPG | Current Version: **`5.13`**
A recreation of the classic EPG on the Orange/Amena Liveboxes and legacy Orange y Canal+/Amena+ from 2005-2010. This is based on the latest
version of the Amena+ EPG, version `5.08.6`.

# Orange Web EPG is now archived
